My Process

Given that no two projects are the same, I'm flexible enough to adapt when the project calls for it. But all in all, my UX process goes like this:

Research

Understanding the problem: define, set goals and research the user.

Analize

Acting on the research: create personas, user stories, flows, sitemaps, etc.

Design

Crafting the data: skeching, mockups, prototypes, branding.

Validate

Testing the solution: repeat previous phases if necessary.
